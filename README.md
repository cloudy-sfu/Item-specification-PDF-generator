# Inventory count
Formatted report generator for inventory images with attributes



## Usage

![image-20260626172021923](./assets/image-20260626172021923.png)

Take a photo or upload from library in tablet or mobile phone. Edit attributes (names, values, units can be customized) and click "Confirm".

![image-20260626171930072](./assets/image-20260626171930072.png)

Edit or delete existed images.

![image-20260626172104374](./assets/image-20260626172104374.png)

Print as a report: A4 paper landscape one image per page.
